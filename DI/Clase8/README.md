# Proyecto DI_UAX

## Descripción
Este es un gestor de tareas en PyQt6 que permite agregar y eliminar tareas.

## Instalación
1. Clona el repositorio:
   ```sh
   git clone https://github.com/tuusuario/DI_UAX.git
   ```


## Ejecutable:

```shell
  pyinstaller --onefile --windowed InterfazPrueba.py
```
